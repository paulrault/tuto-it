Installation de Windows Serveur

Guide étape par étape

Étapes :
  • 1. Télécharger l’ISO
  • 2. Créer une clé USB bootable
  • 3. Démarrer sur la clé USB
  • 4. Choisir la version
  • 5. Installer Windows Server
  • 6. Configurer le mot de passe administrateur

1️⃣ Télécharger l’ISO de Windows Server

Rendez-vous sur le site officiel Microsoft pour télécharger l’ISO.

2️⃣ Créer une clé USB bootable

Utilisez l’outil Rufus pour créer votre clé USB bootable.

3️⃣ Démarrer sur la clé USB

Insérez la clé dans le serveur et redémarrez en choisissant le boot USB.

4️⃣ Choisir la version de Windows Server

Sélectionnez la version correspondant à vos besoins (Standard, Datacenter...).

5️⃣ Installer Windows Server

Suivez l’assistant d’installation en choisissant les options par défaut ou personnalisées.

6️⃣ Configurer le mot de passe administrateur

Définissez un mot de passe sécurisé pour le compte Administrateur.
