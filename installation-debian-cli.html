Installation de Debian CLI

Guide étape par étape

1️⃣ Télécharger l'image ISO

Rendez-vous sur le site officiel de Debian et téléchargez l'image ISO réseau (NetInstall).

2️⃣ Créer un support d'installation

Gravez l'image sur une clé USB ou un DVD bootable.

3️⃣ Démarrer le serveur ou VM

Insérez la clé USB et démarrez la machine sur le support d'installation.

4️⃣ Sélectionner l'installation en CLI

Choisissez l'option “Install” ou “Graphical Install” puis optez pour l'installation minimale CLI si disponible.

5️⃣ Configurer le réseau

Entrez l'adresse IP, masque, passerelle et DNS pour votre serveur.

6️⃣ Partitionner le disque

Utilisez l'option guidée avec LVM ou manuelle selon vos besoins.

7️⃣ Installer les composants de base

Sélectionnez uniquement le système de base, pas d'environnement graphique.

8️⃣ Configurer l'utilisateur et mot de passe root

Créez votre utilisateur et définissez le mot de passe root.

9️⃣ Finaliser l'installation

Terminez l'installation et redémarrez le serveur. Vous êtes prêt à vous connecter en CLI.
