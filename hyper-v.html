Utilisation de Hyper-V

Guide étape par étape

1️⃣ Activer Hyper-V

Activez Hyper-V dans les fonctionnalités Windows ou via PowerShell.

2️⃣ Créer une machine virtuelle

Ouvrez Hyper-V Manager et créez une nouvelle machine virtuelle.

3️⃣ Configurer les ressources

Attribuez la RAM, le CPU et le disque dur à votre VM.

4️⃣ Installer un système d’exploitation

Montez une image ISO et lancez l’installation.

5️⃣ Configurer le réseau

Créez un switch virtuel pour connecter votre VM au réseau.

6️⃣ Gérer la machine virtuelle

Démarrer, arrêter, sauvegarder et cloner vos machines virtuelles.
