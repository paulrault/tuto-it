📁 Gestion des fichiers

Créer un fichier :

touch fichier.txt

Copier :

cp fichier.txt /dossier/

Déplacer :

mv fichier.txt /dossier/

Supprimer :

rm fichier.txt

📂 Navigation

Afficher le dossier courant :

pwd

Lister les fichiers :

ls -l

Changer de dossier :

cd /chemin

⚙️ Système

Mettre à jour :

sudo apt update && sudo apt upgrade

Voir les processus :

top

Redémarrer :

reboot

🌐 Réseau

Tester une connexion :

ping google.com

Afficher IP :

ip a

Afficher ports :

netstat -tuln
