Installation de Windows Serveur

Guide étape par étape

Étapes :
  • 1. Télécharger l’ISO
  • 2. Créer une clé USB bootable
  • 3. Démarrer sur la clé USB
  • 4. Choisir la version
  • 5. Installer Windows Server
  • 6. Configurer le mot de passe administrateur

1️⃣ Télécharger l’ISO de Windows Server

Rendez-vous sur le site officiel de Microsoft pour télécharger l’ISO correspondant à la version de Windows Server souhaitée.

2️⃣ Créer une clé USB bootable

Utilisez un outil comme Rufus pour créer une clé USB bootable avec l’ISO que vous avez téléchargé.

3️⃣ Démarrer sur la clé USB

Insérez la clé USB dans le serveur et configurez le BIOS/UEFI pour démarrer depuis celle-ci.

4️⃣ Choisir la version de Windows Server

Sélectionnez la version qui correspond à vos besoins (Standard, Datacenter, etc.).

5️⃣ Installer Windows Server

Suivez l’assistant d’installation en choisissant les options par défaut ou personnalisées selon votre environnement.

6️⃣ Configurer le mot de passe administrateur

Définissez un mot de passe sécurisé pour le compte Administrateur et terminez la configuration initiale.
